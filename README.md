# 町田市保育園 応募状況ダッシュボード 2026

町田市の認可保育所等の2026年度4月入園一次募集における応募・内定状況を可視化したダッシュボードです。

## デモ

https://machida-hoikuen-stats-2026.pages.dev/

## スクリーンショット

ダッシュボードでは以下の情報を確認できます：

- **統計カード**: 総応募人数、内定人数、内定率、平均応募倍率
- **年齢別応募・内定状況**: 0〜5歳児の応募・内定状況を表形式とグラフで表示
- **地域別応募状況**: 町田市の5つの地域（堺、忠生、町田、南、鶴川）別の応募状況を年齢別に確認可能
- **保育園ランキング**: 応募倍率、応募数、空き数でソート表示
- **保育園検索**: 年齢・地域でフィルタリング、地図表示機能

## 特徴

- 📊 **インタラクティブなグラフ**: Chart.jsを使用した見やすいビジュアライゼーション
- 🗺️ **地図表示**: Leaflet.jsを使用した保育園の位置情報表示
- 🎯 **応募倍率の可視化**: 各保育園の応募倍率を色分けして表示
- 📱 **レスポンシブデザイン**: スマートフォンからPCまで対応
- 🔗 **保育園ホームページへのリンク**: 各保育園の公式サイトへ簡単にアクセス可能

## ローカルでの実行方法

1. リポジトリをクローン
   ```bash

   cd machida-hoikuen-stats-2026
   ```

2. ローカルサーバーを起動

## 重要な注意事項

### 応募倍率について

ダッシュボードに表示される「応募倍率」は、**第一希望から第五希望までの合計応募数**を基に計算されています。そのため、実際の応募者数よりも数値が大きくなっており、**実際の倍率よりも高く表示**されます（例: 実際1.5倍 → 表示7.5倍程度）。

この数値は参考値として使用し、実際の入園難易度を判断する際は、町田市の公式情報も併せてご確認ください。
